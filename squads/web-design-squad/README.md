# web-design-squad

Squad especializado em criar sites e landing pages profissionais e que convertem.

## Agentes

| Agente | Papel |
|--------|-------|
| `@creative-director` | Coordenação, briefing, coerência de marca |
| `@web-designer` | Design visual, layouts, sistema de design |
| `@ux-designer` | UX, personas, fluxos, wireframes |
| `@storyteller` | Brand story, narrativa, tom de voz |
| `@copywriter` | Copy persuasivo, headlines, CTAs |
| `@seo-specialist` | SEO on-page, keywords, meta tags |
| `@frontend-dev` | HTML/CSS/JS, performance, responsividade |
| `@cro-specialist` | Otimização de conversão, A/B, funil |

## Fluxo Recomendado

```
@creative-director *create-brief
  → @ux-designer *create-persona
  → @storyteller *create-brand-story
  → @storyteller *define-tone-of-voice
  → @copywriter *write-headline
  → @copywriter *write-copy
  → @seo-specialist *keyword-research
  → @web-designer *design-landing-page
  → @ux-designer *create-user-flow
  → @frontend-dev *build-page
  → @seo-specialist *optimize-on-page
  → @cro-specialist *audit-landing-page
  → @creative-director *review-brand-consistency
```

## Como Usar

```bash
# Iniciar projeto de site/landing page
@creative-director
*create-brief

# Ou ativar um agente específico
@copywriter
*write-headline
```
