# *define-positioning

Define o posicionamento estratégico da marca para orientar todos os entregáveis do squad.

## Positioning Statement Template

```
Para [público-alvo],
[marca] é a [categoria]
que [benefício principal]
porque [razão para acreditar].

Ao contrário de [concorrentes],
nós [diferencial único].
```

## Brand Pillars

Defina 3-5 pilares que guiam todas as decisões criativas:

```
Pilar 1: [nome] — [descrição em 1 linha]
Pilar 2: [nome] — [descrição em 1 linha]
Pilar 3: [nome] — [descrição em 1 linha]
```
