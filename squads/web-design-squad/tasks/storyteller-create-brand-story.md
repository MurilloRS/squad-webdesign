# *create-brand-story

Cria a narrativa fundacional da marca usando o framework StoryBrand de Donald Miller.

## Framework StoryBrand (7 elementos)

```
1. HERÓI (Cliente)
   Quem é o personagem principal?
   O que ele quer?

2. PROBLEMA
   Externo: obstáculo tangível
   Interno: frustração emocional
   Filosófico: por que isso é injusto?
   Vilão: o que causa o problema?

3. GUIA (Marca)
   Empatia: "Nós entendemos..."
   Autoridade: prova de que podemos ajudar

4. PLANO
   3 passos simples para resolver o problema
   Reduz o risco percebido

5. CHAMADA À AÇÃO
   Direta: "Agende agora"
   Transicional: "Saiba mais"

6. EVITAR O FRACASSO
   O que o herói perde se não agir?
   Urgência emocional

7. ALCANÇAR O SUCESSO
   Transformação desejada
   Vida melhor pós-produto
```

## Template de Brand Story

```
[MARCA] entende que [HERÓI] quer [OBJETIVO].
Mas [PROBLEMA EXTERNO], o que faz com que [PROBLEMA INTERNO].

Por isso, criamos [SOLUÇÃO].
Como [GUIA COM AUTORIDADE], ajudamos [HERÓI] a [PLANO EM 3 PASSOS].

Sem [MARCA], [HERÓI] continua [SITUAÇÃO DE FRACASSO].
Com [MARCA], [HERÓI] finalmente [TRANSFORMAÇÃO].
```
