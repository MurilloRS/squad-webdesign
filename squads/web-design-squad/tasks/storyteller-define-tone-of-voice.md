# *define-tone-of-voice

Define a personalidade verbal da marca — como ela fala, não apenas o que fala.

## Framework de Tom de Voz

### 4 Dimensões

```
1. FORMAL ←————————→ CASUAL
   Onde a marca se posiciona neste espectro?

2. SÉRIO ←————————→ DIVERTIDO
   Que nível de humor/leveza é adequado?

3. RESPEITOSO ←————————→ IRREVERENTE
   Quão ousada e desafiadora é a marca?

4. ENTUSIASTA ←————————→ PRAGMÁTICO
   Quanto de energia e emoção vs. fatos diretos?
```

### Template de Guia

```markdown
## A [MARCA] é:

**[Adjetivo 1]** — mas não [extremo negativo]
Exemplo certo: "[frase exemplo]"
Exemplo errado: "[frase exemplo]"

**[Adjetivo 2]** — mas não [extremo negativo]
Exemplo certo: "[frase exemplo]"
Exemplo errado: "[frase exemplo]"

## Vocabulário Preferido
- Use: [palavra1], [palavra2], [palavra3]
- Evite: [palavra1], [palavra2], [palavra3]

## Nunca diga:
- [expressão genérica a evitar]
- [clichê do setor]
```
